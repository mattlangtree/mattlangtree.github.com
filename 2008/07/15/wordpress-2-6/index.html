Wordpress Releases Feature-rich 2.6 Version of Blogging Software (Updated X2)

Jul 15th, 2008

The new version 2.6 release from the folks at Wordpress comes as a suprise, arriving a month ahead of schedule.

The new blogging software touts the following new features:

  • Post Revisions: Wiki-like tracking of edits - Press This! Button (now extracts youtube video code/flickr source for easy image/video embedding in posts) - Google Gears support: Speed up your blogging experience by caching Javascript/CSS on your local computer - Theme Previews: Preview a theme running on your site - without having the public blog appearance change
    Some other new features, which I’m glad to see are: - Full SSL support - Image Captions - Custom wp-config.php location - for “clean” svn checkouts.

The press release is available here: http://wordpress.org/development/2008/07/wordpress-26-tyner/

Thanks to Adam Wolfs, for the heads up about this new release!

UPDATE: I’ve installed Wordpress 2.6 on this domain, using the SVN installation method. I’m happy to announce that the process for an SVN installation is mostly trivial and the site was only down for about 30 seconds.

UPDATE 2: The admin area does seem to be a little snappier at times, after installing Google gears on Firefox 3.
